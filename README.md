# คู่มือการปรับแต่งเว็บไซต์ (ร้านซ่อมคอมพิวเตอร์)

ไฟล์ที่สร้างให้:
- `index-new.html` — หน้าแรก (ถ้าต้องการให้เป็นหน้าเริ่มต้น ให้เปลี่ยนชื่อเป็น `index.html`)
- `about.html` — เกี่ยวกับเรา
- `services.html` — รายการบริการ
- `contact.html` — ติดต่อ (มี iframe แผนที่ ให้เปลี่ยน src เป็นตำแหน่งร้านของคุณ)
- `styles.css` — สไตล์ของเว็บไซต์

สิ่งที่ควรปรับให้เป็นข้อมูลจริง:
- เบอร์โทร: ค้นหาและแก้ `<a href="tel:+66000000000">0-0000-0000</a>` ในไฟล์ทั้งหมด
- LINE: แก้ลิงก์ `https://line.me/ti/p/@yourlineid` เป็นไอดีร้านของคุณ และเพิ่ม QR (รูป) ถ้ามี
- ที่อยู่: แก้ข้อความที่อยู่บน `contact.html`
- แผนที่: ไปที่ Google Maps → Share → Embed a map → คัดลอก src ของ iframe มาแทนที่ค่าใน `contact.html`
- โลโก้/ภาพประกอบ: แทนที่โลโก้ตัวอักษรในส่วน `.logo` ด้วยรูปจริง (วางไฟล์ใน `assets/` และแก้ `<img src=...>`)

การเผยแพร่ (ง่าย):
- อัปโหลดทั้งโฟลเดอร์นี้ไปที่โฮสติ้ง (เช่น GitHub Pages, Netlify, Vercel หรือโฮสติ้งที่ใช้) โดยหน้าเริ่มต้นควรเป็น `index.html`

คำแนะนำเพิ่มเติม:
- หากต้องการให้ผมช่วยเปลี่ยน `index.html` ที่มีอยู่เป็นเวอร์ชันใหม่ ให้บอกว่าต้องการสำรองไฟล์เดิมไว้หรือให้แทนที่ทันที
- ผมสามารถเพิ่มฟีเจอร์เพิ่มเติมได้ เช่น ฟอร์มติดต่อ (ส่งอีเมล), ไฟล์รูปภาพตัวอย่าง, หรือ QR Code สำหรับ LINE

ขอให้ธุรกิจของคุณเติบโตและได้ลูกค้าพึงพอใจมากขึ้น!
